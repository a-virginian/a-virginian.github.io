how to open and close vim

to open vim type vim and then the fille you want to open like this: vim file_name. there are several ways to close vim for all of them you start by clicking escape then typing colon. one way to close it once that is done you can type q this will quit vim without saving to quit and save you type wq

some basic controls in vim

J
moves down one row
k
moves up a row
L
moves to the right when on a row with text
H
moves to the left when on a row with text
i
puts you into insert mode which allows you to edit the page.
syntax on
to put on syntax which will color code you code you first type escape then : syntax on
w
if you want to save you can click escape then : and lastly w
