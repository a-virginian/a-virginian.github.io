How to Use Vim

One thing i have learned during web page design is how to use vim.

How to Open and Close Vim

how to open vim

  1. Navigate to the file you wan to edit with vim.
  2. Open the file that you want edit by typing vim then the file name like this: vim file_name.

There are several ways to close vim for all of them you start by clicking escape then :. One way to close it once that is done you can type q this will quit vim without saving. To quit and save you type wq .

some basic controls in vim

J
Moves down one row
k
Moves up a row
L
Moves to the right when on a row with text
H
Moves to the left when on a row with text
i
Puts you into insert mode which allows you to edit the page.
syntax on
To put on syntax which will color code you code you first type escape then : syntax on
w
If you want to save you can click escape then : and lastly w
