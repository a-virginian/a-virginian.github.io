How to Use Vim

One thing I have learned during web page design class is how to use vim.

How to Open and Close Vim

how to open vim

  1. open the termianl
  2. Navigate to the file you want to edit.
  3. Open the file by typing "vim" then the file name, like this: vim file_name.
  4. hit Enter or Return

There are several ways to close vim. For all of them, you start by clicking escape, then :. Next you type one of the 2 following things:

  • q, This quits vim without saving.
  • wq, This save and quits.
J
Moves down one row
k
Moves up a row
L
Moves to the right when on a row with text
H
Moves to the left when on a row with text
i
Puts you into insert mode which allows you to edit the page.
syntax on
To put on syntax which will color code you code you first type escape then : syntax on
w
If you want to save you can click escape then : and lastly w
